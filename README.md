# memo

Personal memorandum related to work

## require

Go

## setup

- install hugo

- win or linux

```
git clone https://github.com/gohugoio/hugo.git
cd hugo
go install
```

- macOS

```
brew install hugo
```

- install theme

```
git submodule update
```


## serve

```
hugo server -D
```

## generate static site pages

```
hugo -D
```
